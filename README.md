# QR-Based Smart Attendance System 📱✅

A complete, end-to-end software engineering project designed to modernize and streamline the attendance tracking process using QR code technology. This repository contains not only the source code but also the complete Software Development Life Cycle (SDLC) documentation.

### 🎯 Project Overview
This system provides a fast, secure, and reliable way to record attendance. By scanning dynamically generated QR codes, it eliminates manual entry errors and saves significant time in educational or corporate environments.

### 📐 System Architecture & Documentation
I approach software development with a strong focus on architectural design, careful planning, and user experience. This project includes comprehensive engineering artifacts:
- **Requirement Analysis:** Detailed documentation of functional and non-functional requirements.
- **System Design (PlantUML):** Architectural diagrams and use-case scenarios mapping out the system's logic and data flow.
- **Usability Testing:** Structured test scenarios and user feedback integration to ensure a seamless interface.
- **User Manual:** Complete guidelines for end-users to navigate the system effectively.
